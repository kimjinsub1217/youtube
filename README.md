# 📌 youtube
> 유튜브

## ✔기능
<ul>
  <li>영상 목록을 클릭하여 ExoPlayer를 사용해 오디오 및 동영상 재생이 가능함  </li>
  <li>Retrofit 을 이용해 영상 목록을 받아와 구성함 </li>
  <li>MotionLayout을 이용해 유튜브 영상 플레이어 화면 전환 애니메이션을 구현함  </li>

</ul>

##  ✔활용 기술
MotionLayout, ExoPlayer, Retrofit
